    Get Started¶

    1. Prerequisites¶

    1. Install the Nodejs

    2. Install the git

    3. install the VSCode

    4. Install mysql or posgresql database

    5. Install the following npm modules globally

      1. typescript
      2. tslint
      3. rimraf 
      1
      npm i typescript tslint rimraf mission.cli -g

    2. Verify the installatioin¶

    1. tsc -v
    2. tslint -v
    3. node -v
    4. git version
    5. mio version
